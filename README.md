# 车票信息自动获取应用

## 项目简介

这是一款使用Python开发的桌面端车票信息自动获取应用，支持从12306网站抓取车票信息，提供定时查询、结果导出等功能。

## 功能特点

- **多种车次类型支持**：支持高铁、动车、普通列车等多种车次类型的信息获取
- **自定义查询参数**：可设置出发地、目的地、日期等关键参数
- **定时自动查询**：支持设置查询频率和时间范围
- **自动盯票功能**：支持自动盯直达票，当发现余票时发送通知
- **邮箱提醒功能**：支持通过邮箱发送余票通知，可配置是否记住邮箱信息
- **查询用时显示**：每次查询完毕后显示查询用时，方便用户了解查询效率
- **夜晚模式切换**：支持白天/夜晚模式切换，减少眼睛疲劳
- **软件记忆功能**：自动记住用户退出时的设置，包括出发地、目的地、日期、车次类型、自动盯票配置和夜晚模式设置
- **健壮的反爬机制**：包括合理的请求间隔设置、User-Agent随机切换等
- **直观的用户界面**：使用PyQt开发，布局清晰，操作简单
- **查询结果导出**：支持导出为Excel和CSV格式
- **完善的日志记录**：便于问题排查和维护

## 技术架构

- **开发语言**：Python 3.7+
- **GUI框架**：PyQt5
- **网络请求**：requests
- **网页解析**：BeautifulSoup4 + lxml
- **定时任务**：schedule
- **数据处理**：pandas
- **日志记录**：logging

## 使用方法

### 1. 运行应用

```bash
python main.py
```

### 2. 设置查询条件

- **出发地**：输入出发城市名称（如：北京）
- **目的地**：输入到达城市名称（如：上海）
- **出发日期**：选择出发日期
- **车次类型**：选择车次类型（全部、高铁、动车、普通列车）

### 3. 执行查询

- **立即查询**：点击按钮执行单次查询，获取直达车次信息
- **查询中转车次**：点击按钮查询中转车次信息
- **自动盯票**：点击按钮配置并启动自动盯票功能
  1. **盯票功能说明**：
     - 只能盯直达票，中转票不支持
     - 自动查询间隔不能低于30秒
     - 当盯到票后，自动停止盯票
  2. **邮箱提醒配置**：
     - 启用邮箱提醒后，当发现余票时会发送邮件通知
     - 输入邮箱地址和邮箱授权码（注意：是授权码，不是登录密码）
     - 可选择是否记住邮箱配置，下次打开时会自动填充
  3. **车类型和座位等级选择**：
     - 可选择需要盯的车类型（高铁、动车、普通列车）
     - 可选择需要盯的座位等级（一等座、二等座、硬卧等）
     - 可选择具体的车次进行盯票

### 4. 导出结果

- **导出Excel**：将查询结果导出为Excel文件
- **导出CSV**：将查询结果导出为CSV文件

### 5. 清空结果

- **清空结果**：点击按钮清空当前查询结果

### 6. 软件记忆功能

- **自动保存设置**：软件会自动记住用户退出时的设置，包括：
  - 出发地和目的地
  - 出发日期
  - 车次类型
  - 自动盯票配置
  - 邮箱提醒配置（如果选择了记住）
- **下次启动自动加载**：下次打开软件时，会自动显示用户上一次退出时的设置

## 注意事项

1. 本应用仅用于学习和研究目的，请勿用于商业用途
2. 使用时请遵守12306网站的使用条款和相关法律法规
3. 频繁查询可能会被网站限制，建议合理设置查询间隔
4. 自动盯票功能的查询间隔不能低于30秒，以防止对12306服务器造成负担
5. 邮箱提醒功能需要使用邮箱授权码，而不是登录密码
6. 不同邮箱的授权码获取方式可能不同，请参考对应邮箱的帮助文档
7. 如遇网页结构变化，可能需要更新解析规则
8. 自动盯票功能仅支持直达票，中转票不支持

## 故障排查

1. **网络连接失败**：检查网络连接是否正常，防火墙是否阻止了请求
2. **解析失败**：可能是12306网站结构发生了变化，需要更新解析规则
3. **导出失败**：检查目标文件夹是否有写入权限
4. **自动盯票不执行**：检查是否有其他程序占用了系统资源，或者网络连接是否稳定
5. **邮箱提醒失败**：检查邮箱地址和授权码是否正确，邮箱是否开启了SMTP服务，网络连接是否稳定
6. **查询间隔设置失败**：自动盯票的查询间隔不能低于30秒，请调整为30秒或以上
